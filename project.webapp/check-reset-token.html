Check Your Reset Token

What This Tool Does

This tool will check if your reset token is:

  • Valid - Token exists and is not expired
  • Expired - Token exists but has passed its expiration time
  • Used - Token was already used to reset password
  • Invalid - Token doesn't exist in database

What to Do Based on Results

If token is Valid: You can use it to reset your password

If token is Expired/Used/Invalid: You need to request a new password reset
